<h1>Tab Navigation Component</h1>

<p>This project is a simple and reusable<b>Tab Navigation UI</b> built using <b>HTML, CSS and JavaScript.</b> Users can click on different tabs and the active tab will show its related content while hiding the rest.</p>

<h2>Features</h2>

<ul>
    <li>Multiple tabs in one page</li>
    <li>Only one tab shows at a time</li>
    <li>Active tab style highlight</li>
    <li>Smooth tab switching without page reload</li>
    <li>Add a smooth underline animation under the active tab</li>
</ul>

<h2>Technologies Used</h2>

<ul>
    <li>HTML -> Tab layout & content sections</li>
    <li>CSS -> Styling & active tab highlighting</li>
    <li>JavaScript -> Switching visible tab content</li>
</ul>

<h2>How It Works</h2>

<ul>
    <li>Each tab has a 'data-tab' attribute</li>
    <li>On click -> JS reads which tad user selected</li>
    <li>That tab becomes 'active' and its related content becomes visible</li>
    <li>All other content blocks are hidden</li>
</ul>

<h2>Future Improvements</h2>

<ul>
    <li>Add swipe support for mobile</li>
    <li>Support multiple themes (light/dark)</li>
    <li>Load tab data dynamically using JSON</li>
</ul>
