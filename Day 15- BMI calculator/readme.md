<h1>BMI Calculator</h1>

<p>This project is a simple and interactive <b>BMI Calculator</b> built using <b>HTML, CSS and JavaScript.</b> Users can enter their height and weight, and the calculator instantly displays their BMI along with category results (Underweight, Normal, Overweight, Obese).</p>

<h2>Features</h2>

<ul>
    <li>Calculates BMI instantly</li>
    <li>Displays BMI category with colors</li>
    <li>Clean and responsive UI</li>
    <li>Error handing for empty or invalid inputs</li>
    <li>Ideal beginner-friendly JS logic</li>
</ul>

<h2>Technologies Used</h2>

<ul>
    <li>HTML -> Form inputs & result section</li>
    <li>CSS -> Layout design, result highlighting</li>
    <li>JavaScript -> BMI formula, validation & category logic</li>
</ul>

<h2>How It Works</h2>

<ol>
    <li>User enters height (in cm) and weight (in kg)</li>
    <li>Result is displayed along with a category:
    <ul>
    <li>Underweight</li>
    <li>Normal</li>
    <li>Overweight</li>
    <li>Obese</li>
    </ul>
    </li>
    <li>CSS highlights the category using different colors</li>
</ol>

<h2>Formula Used</h2>

<p>BMI = weight (kg)/(height (m))^2</p>
