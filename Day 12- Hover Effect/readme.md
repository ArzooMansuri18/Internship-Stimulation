<h1>Mouse Hover Portfolio Card</h1>

<p>This project is a stylish <b>Portfolio Card with Hover Effects</b> made with <b>HTML, CSS and JavaScript.</b> When the user hovers over the card, smooth animations reveal additional details, creating a modern interactive UI component.</p>

<h2>Features</h2>

<ul>
    <li>Smooth hover animation</li>
    <li>Image zoom or fade-in effect</li>
    <li>Text/content reveal on hover</li>
    <li>Fully responsive card design</li>
    <li>Clean and modern UI perfect for portfolios</li>
</ul>

<h2>Technologies Used</h2>

<ul>
    <li>HTML -> Card Structure: image + content</li>
    <li>CSS -> Hover animations, transitions, shadows</li>
    <li>JavaScript -> (Optional) to add extra interactivity</li>
</ul>

<h2>How It Works</h2>

<ul>
    <li>A card contains an image and hidden content section</li>
    <li>Using CSS hover pseudo-class(:hover), animations are triggered: <ul> <li>Image zoom effect</li> <li>Text fading in or sliding up</li> <li>Shadow enhancement</li> </ul></li>
    <li>JavaScript can optionally be used for extra effects like dynamic content loading.</li>
</ul>

<h2>Future Improvements</h2>

<ul>
    <li>Add flipping card effect</li>
    <li>Add modal pop-up for project details</li>
    <li>Add animations on scroll using Intersection Observer</li>
</ul>
