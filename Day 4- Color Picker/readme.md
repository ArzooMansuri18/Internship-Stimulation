<h1>Tab Navigation Component</h1>

<p>This project is a simple and interactive <b>Color Picker Tool</b> built using <b>HTML, CSS and JavaScript.</b>It allows the user to pick a color using an input color picker and instantly see the selected color applied to a preview box.</p>

<h2>Features</h2>

<ul>
    <li>Select any color using native color input</li>
    <li>Live preview of the selected color</li>
    <li>Hex value updates automatically</li>
    <li>Minimal and clean UI</li>
</ul>

<h2>Technologies Used</h2>

<ul>
    <li>HTML -> Page structure + color input element</li>
    <li>CSS -> Styling & layout</li>
    <li>JavaScript -> Handle color change and UI updates</li>
</ul>

<h2>How It Works</h2>

<ul>
    <li>The user picks a color from the color input</li>
    <li>JS listens to the 'input' or 'change' event</li>
    <li>When user selects a color: 
    <ul> <li>Preview box background color updates instantly</li> 
    <li>Color code text also updates</li> </ul> </li>
</ul>

<h2>Possible Future Improvements</h2>

<ul>
    <li>Support RGB/HSL values</li>
    <li>Save favourite colors list</li>
    <li>Support light/dark mode theme</li>
</ul>

<h2>Folder Structure</h2>

<pre>/color-picker-tool
|->index.html
|->style.css
|->main.js</pre>

<h2>How to Run</h2>

<ul>
    <li>Open 'index.html' in your browser</li>
    <li>Pick any color from the color input</li>
    <li>See the background of preview box update instantly</li>
</ul>
