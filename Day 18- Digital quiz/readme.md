<h1>Digital Quiz App</h1>

<p>This project is a fully functional <b>Digital Quiz Application</b> built using <b>HTML, CSS and JavaScript.</b> Users can answer multiple-choice questions, see their score at the end, and restart the quiz anytime.</p>

<h2>Features</h2>

<ul>
    <li>Multiple-choice questions</li>
    <li>Naxt question logic</li>
    <li>Shows final score at the end</li>
    <li>"Restart Quiz" functionally</li>
    <li>Highlight correct and wrong answers (optional)</li>
</ul>

<h2>Technologies Used</h2>

<ul>
    <li>HTML -> Question container, options, buttons</li>
    <li>CSS -> Layout, card design, button styling</li>
    <li>JavaScript -> Quiz logic, scoring, DOM updates</li>
</ul>

<h2>How It Works</h2>

<ul>
    <li>A JAvaScript array stores all quiz questions</li>
    <li>When the quiz starts: 
    <ul> 
        <li>The first question loads on screen</li> 
        <li>User selects an option</li> 
        <li>Clicking "Next" moves to the next question</li> 
    </ul> 
    </li>
    <li>When all questions are answered: 
    <ul>
        <li>Score page is shown</li>
        <li>Correct answers count displayed</li> 
        <li>Restart button allows quiz to begin again</li> 
    </ul>
    </li>
</ul>

<h2>Future Improvements</h2>

<ul>
    <li>Add timer for each question</li>
    <li>Add negetive marking</li>
    <li>Save score in localStorage</li>
</ul>
