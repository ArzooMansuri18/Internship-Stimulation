<h1>Tab Navigation Component</h1>

<p>This project is a simple and modern custom <b>Drop-down Menu UI</b> built using <b>HTML, CSS and JavaScript.</b>It replaces the boring default browser <select> style with a stylish, interactive drop-down element.</p>

<h2>Features</h2>

<ul>
    <li>Fully custom UI (no default select tag)</li>
    <li>Click to open/close options</li>
    <li>Click outside to auto-close menu</li>
    <li>Selected item shows as active</li>
    <li>Smooth interaction with vanilla JavaScript</li>
</ul>

<h2>Technologies Used</h2>

<ul>
    <li>HTML -> Dropdown structure</li>
    <li>CSS -> Styling & animation</li>
    <li>JavaScript -> Toggle menu + select option</li>
</ul>

<h2>How It Works</h2>

<ul>
    <li>Clicking the main dropdown box toggles the option list</li>
    <li>Each option has a click listener</li>
    <li>When user selects any option: <ul> <li>The selected text updates in the main box</li>
    <li>Options list closes automatically</li> </ul></li>
    <li>Clicking anywhere outside the closes the dropdown as well</li>
</ul>

<h2>Future Improvements</h2>

<ul>
    <li>Add keyboard navigation (arrow up/down)</li>
    <li>"Selected option" text inside the button once link is clicked</li>
    <li>Add search filtering for large option lists</li>
</ul>
