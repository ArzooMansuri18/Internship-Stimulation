<h1>Dynamic News Ticker</h1>

<p>This project is a smooth <b>News Ticker</b> built using <b>HTML, CSS and JavaScript.</b> The text scrolls from right to left continuously, just like breaking-news tickers seen on news channels and websites.</p>

<h2>Features</h2>

<ul>
    <li>Continuous scrolling animation</li>
    <li>Smooth and seamless loop</li>
    <li>Easy to update the news items</li>
    <li>Fully responsive ticker bar</li>
    <li>Adjustable speed (via JavaScript)</li>
</ul>

<h2>Technologies Used</h2>

<ul>
    <li>HTML -> Text container + ticker wrapper</li>
    <li>CSS -> Animation using @keyframes + styling</li>
    <li>JavaScript -> Change text dynamically (optional)</li>
</ul>

<h2>How It Works</h2>

<ul>
    <li>A wrapper contains all the news items in a single line</li>
    <li>CSS animation moves the text horizontally(translateX)</li>
    <li>Animation loops infinitely to create a continuous scrolling effect</li>
    <li>JavaScript may be used to: 
    <ul> 
    <li>Add new headlines dynamically</li> 
    <li>Changes speed</li> 
    <li>Shuffle news items</li> 
    </ul> </li>
</ul>

<h2>Future Improvements</h2>

<ul>
    <li>Fetch news dynamically from an API</li>
    <li>Add pause-on-hover</li>
    <li>Add fade-in & fade-out effects</li>
    <li>Add direction toggle (left -> right or right -> left)</li>
</ul>
