<h1>Tip Calculator</h1>

<p>This project is a lightweight and interactive <b>Tip Calculator</b> built using <b>HTML, CSS and JavaScript.</b> It helps users calculate the tip amount and total bill per person - useful for restaurants, cafes, and group outings.</p>

<h2>Features</h2>

<ul>
    <li>Calculate tip based on percentage</li>
    <li>Shows total amount per person</li>
    <li>Error handling for empty fields</li>
    <li>Immediate updates as user changes inputs</li>
    <li>Responsive design</li>
</ul>

<h2>Technologies Used</h2>

<ul>
    <li>HTML -> Bill input, tip selection, result display</li>
    <li>CSS -> Layout, spacing, visual styling</li>
    <li>JavaScript -> Core tip calculation</li>
</ul>

<h2>How It Works</h2>

<ul>
    <li>User enters: 
    <ul>
        <li>Bill amount</li>
        <li>Tip percentage</li>
        <li>Number of people</li>
    </ul>
    </li>
    <li>JavaScript calculates: 
    <ul>
        <li>Tip amount </li>
        <li>Total bill including tip</li>
        <li>Per-person share</li>
    </ul>
    </li>
    <li>Results update dynamically when input values changes</li>
</ul>

<h2>Validation Added</h2>

<ul>
    <li>Prevent division by zero</li>
    <li>Prevent negative values</li>
    <li>Shows message when required fields are empty</li>
</ul>

<h2>Future Improvements</h2>

<ul>
    <li>Add tip percentage buttons (5%, 10%, 15%, 20%)</li>
    <li>Add slider for tip selection</li>
    <li>Add currency symbol selector</li>
</ul>
